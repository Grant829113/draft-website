Final Project


For my final project in CSC 106, I created a "Space Invader" type app where the user can shoot falling rocket objects with the user object. The movement of the user object is controlled by the mouse and the user can shoot by clicking. The goal is to not have the falling objects pass or hit you,or else you lose points. The user gains points by shooting the rockets. The user wins by reaching 100 points.
